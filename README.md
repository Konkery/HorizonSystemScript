## HORIZON SERVER SYSTEM SCRIPT

Репозиторий содержит ПО и скрипты, предназначенные для первоначальной настройки программного обеспечения, входящего в состав платформы автоматизации `Horizon Automation`. А именно — ее серверной составляющей, работающей на аппаратном обеспечении `HUB` (на базе промышленных микрокомпьютеров Raspberry Pi CM4), и информационных компонентов платформы.

### Директории репозитория

1. **js** — содержит исходный код модулей на JavaScript (Node.js) и системные скрипты Bash;
2. **res** — содержит вспомогательные файлы: конфигурации, инструкции, схемы и переменные окружения.

---

### Состав директории `js\app`

#### 1. LinuxScript
Содержит конфигурации системных служб `systemd` и скрипты для настройки ОС Linux:
*   `nodered1.service` – `nodered5.service` — службы для запуска независимых инстансов Node-RED под разными пользователями (`operator1-5`);
*   `pb.sh` — скрипт настройки безопасного завершения работы ОС при нажатии физической кнопки питания (использует `triggerhappy` и `logind`);
*   `README-LinuxScript.md` — подробное описание процедур настройки Linux-окружения.

#### 2. MongoDBscript
Модули для инициализации и настройки СУБД MongoDB:
*   `01__MongoDB-CreateUserAdmin-rev.01-v01.js` — создание корневого администратора (`root`);
*   `02__MongoDB-InitParam-rev.02-v04.js` — системная настройка: разрешение удаленного доступа и включение авторизации;
*   `03__MongoDB-CreateUserAndCollection-rev.01-v05.js` — создание прикладной структуры: баз данных, пользователей и специальных коллекций (TimeSeries, Capped);
*   `README-MongoDBscript.md` — пошаговая инструкция по развертыванию БД.

#### 3. NodeRedScript
Инструменты для управления средой потокового программирования Node-RED:
*   `master_deploy.js` — оркестратор (запуск с ПК инженера): автоматическая генерация хешей паролей и удаленное обновление `settings.js` на Хабах через SSH;
*   `setup_master_keys.js` — утилита для массовой рассылки публичных SSH-ключей инженера по аккаунтам Хаба для беспарольного доступа;
*   `sync_auth.js` — скрипт локальной синхронизации учетных записей Node-RED с базой данных (запуск на Хабе);
*   `README-NodeRedScript.md` — руководство по управлению аутентификацией и развертыванию инстансов.

---

### Состав директории `res`

1. **01__документация** — подробные инструкции по применению утилит репозитория;
2. **02__mongodb-config** — эталонные конфигурационные файлы (`mongod.conf`);
3. **03__node-red-config** — шаблоны настроек среды исполнения (`settings.js`).
4. **.env** — файл переменных окружения с проектными константами.
